# Shalin Madabhavi's Portfolio

A clean, minimal portfolio website built with Next.js 14 App Router, Tailwind CSS, and Shadcn UI.

## ✨ Features

- Minimal design with dark/light mode toggle
- Working contact form (sends real emails via Resend)
- Responsive mobile-first design
- Projects showcase
- Clean typography & smooth animations

## 🛠️ Tech Stack

**Frontend:**
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Shadcn UI
- Radix UI

**Backend / Infra:**
- Resend (Email)
- Vercel (Hosting)
